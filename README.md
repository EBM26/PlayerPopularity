##PlayerPopularity

-- [See it in production!](http://playerproductivity.herokuapp.com/players)

This is an app that tracks NBA players popularity by the amount of tweets the player recieves daily.

###This app uses:

-Ruby on Rails

-Tweeter API

-PostgreSQL

-Bourbon

-Pure CSS
